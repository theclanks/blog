Exportando seu projeto no GIT para tar.bz2

 Qua 10 outubro 2012 by The Clanks

Para exportar seu projeto basta executar o comando abaixo na pasta raiz do projeto:

git archive --format tar master | bzip2 > ../../../projeto.tar.bz2
