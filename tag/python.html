Solução simples para descompilar arquivos pyc

cd pycdc make
./bin/pycdc Example.pyc > Example.py

Git no ssh do Openshift
Seg 10 setembro 2012 by The Clanks

Para utilizarmos o Git direto no servidor Openshift, precisamos trabalhar com as variáveis:

  • GIT_DIR “Diretório do git”
  • GIT_WORK_TREE “Diretório raiz do projeto”

Você pode exporta-las para economizar comandos:

$ export GIT_WORK_TREE=$HOME"/blog/repo/"
$ export GIT_DIR=$HOME"/git/blog.git/"
