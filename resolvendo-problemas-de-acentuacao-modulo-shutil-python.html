Resolvendo problemas de acentuação módulo Shutil - Python

Qui 20 setembro 2012 by The Clanks

Após muito sofrimento e horas de tortura com um problema de acentuação, encontrei uma solução válida para utilização do módulo shuitil com arquivos que possuem acentução. Utilizando a função .encode("utf-8")

Segue o Erro gerado antes da função:

  • UnicodeDecodeError: 'ascii' codec can't decode byte
import shutil
    shutil.move(origem, destino.encode("utf-8"))
